Learn what a musical key is and how different collections of notes create different keys.

A musical key is a specific collection of notes. When the notes in the collection change, the key changes. You might think of it like a recipe - when the ingredients change, so does the recipe! As a result, there are many different keys in music.

Recap

  • Each key has a unique name starting with a letter
  • Key names use letters from the musical alphabet
  • Keys can be major or minor (D major, D minor)
  • Major and minor keys have different sounds

Learn how keys are named using letters from the musical alphabet, and the difference between major and minor keys.

Each key has a unique name starting with a letter from our musical alphabet. For example, D major or D minor. Major keys often sound bright and happy. Minor keys often sound darker or more serious.
