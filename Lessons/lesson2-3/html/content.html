Title: Rests

Overview: Every note has a silent partner called a “rest”. When we rest in music, we pause and stay silent until the rest is over. There are four kinds of basic rests: Whole rest, half rest, quarter rest, and eighth rest. (Each rest should have an image with it). A whole rest is worth four beats. When we are counting we hold for 1 - 2 - 3 - 4. A half rest is worth two beats. When we are counting we hold for 1 - 2 - 3 - 4. A quarter rest is worth one beat. When we are counting we hold for 1 - 2 - 3 - 4. An eighth rest is worth a half beat. When we are counting we hold for ½ - 1 - 2 - 3 - 4. A trick to tell the difference between a half and whole rest is that a half rest looks like a hat. The whole rest looks like a hole in the ground (show both side by side).
