Recap

  • Musical alphabet: A to G, then repeats
  • Each repeat is higher pitched
  • Press finger on string = next note
  • Three tapes mark finger positions

Learn how the musical alphabet works and how pressing fingers on strings creates different notes.

The notes on each string follow the musical alphabet. The musical alphabet starts at A and ends at G, then it loops back to A! Every time we go through the musical alphabet, the notes become higher pitched. Every time we press down on a string, the note changes into the next note in the musical alphabet.
