Title: Key Signatures

Overview: Next to the treble clef on the staff we have a space for a key signature. A key signature is a set of up to seven sharps or flats. We can’t mix and match sharps and flats in the key signature. The order of sharps that appear in key signatures is set, and matches the phrase “Fluffy Cats Go Down Alleys Eating Birds”. The order of flats is also set, and matches the phrase “Ben Ellie And Dad Go Catch Fish”. We will only focus on the major key signatures for these lessons. (Show an image of a staff with the G Major Key Sig). This is the G Major Key Signature, it has one sharp on the top line. This means that all F notes are sharp. Every time we add a sharp to the key signature, we make a new note sharp and this changes the key.
